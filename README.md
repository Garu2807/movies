# Movies

Небольшое React‑приложение для просмотра каталога фильмов с поиском, фильтрами и избранным. Данные берутся из локального `json-server`.

## Возможности

- список фильмов с карточками и страницей деталей
- поиск по названию
- фильтры по жанру, году и рейтингу
- добавление фильмов в избранное (хранится в `localStorage`)

## Технологии

- React 19 + TypeScript
- Vite
- React Router
- TanStack Query
- Redux Toolkit
- json-server

## Требования

- Node.js 18+ (рекомендуется)
- npm

## Запуск

1) Установить зависимости:

```bash
npm install
```

2) Запустить приложение и API одновременно:

```bash
npm run dev
```

После запуска:

- фронтенд: `http://localhost:5173`
- API: `http://localhost:4000/movies`

## Данные

Фильмы находятся в `db.json`. Можно редактировать файл и перезапустить `json-server`.
